HTML Tags Description
Tag Name Description
<!DOCTYPE html> !DOCTYPE HTML This lets the browser know what version of HTML to expect.
<html></html> HyperText Markup Language Contain all of your HTML code.
<head></head> Head Here include page name, metadata, link to favicon, link to CSS local file or external.
<title></title> Title To add a tittle to Web Site. That is shown in a browser´s title bar.
<link href='source' rel='stylesheet'> Link Use to link with external documents like CSS file or favicon.
<a href="mot.html"></a> Anchor Create a hyperlink to web pages, files, email address, locations in or out of the page.
<!-- --> Comment Represents textual notations and they are avaible to read from source code.

CSS Tags Description
Tag Name Description
font-weight font-weight Sets the weight (or boldness) to the font.
font-style font-style Sets whether a font should be styled with a normal, italic, or oblique face from its font-family.
line-height line-height It's commonly used to set the distance between lines of text.
word-spacing word-spacing Sets the length of space between words and between tags.
letter-spacing letter-spacing Sets space between individual letter.
